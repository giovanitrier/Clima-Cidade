# 🌦️ Clima Cidade

Aplicação web desenvolvida para consulta de previsão do tempo atual e dos próximos 14 dias, utilizando geolocalização do navegador e integração com APIs externas de clima. O sistema identifica automaticamente a localização do usuário (com fallback para uma cidade padrão) e permite também a busca manual por cidades, exibindo dados climáticos de forma clara e organizada.

## 🚀 Tecnologias utilizadas

### Front-end
- Angular (Standalone Components)
- TypeScript
- Bootstrap
- HTML5 e CSS3
- Consumo de APIs REST com HttpClient

### Back-end
- Java
- Spring Boot
- Spring Web (REST API)
- Integração com API Open-Meteo

### APIs externas
- Open-Meteo Forecast API (clima atual e previsão de 14 dias)
- Geolocation API do navegador

## 🔧 Funcionalidades

- Clima atual com data e horário
- Previsão detalhada para os próximos 14 dias
- Detecção automática da localização do usuário
- Busca manual por cidade
- Interface responsiva e intuitiva
